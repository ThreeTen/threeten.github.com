43  /**
44   * A clock providing access to the current instant, date and time using a time-zone.
45   * <p>
46   * Instances of this class are used to find the current instant, which can be
47   * interpreted using the stored time-zone to find the current date and time.
48   * As such, a clock can be used instead of {@link System#currentTimeMillis()}
49   * and {@link TimeZone#getDefault()}.
50   * <p>
51   * Use of a {@code Clock} is optional. All key date-time classes also have a
52   * {@code now()} factory method that uses the system clock in the default time zone.
53   * The primary purpose of this abstraction is to allow alternate clocks to be
54   * plugged in as and when required. Applications use an object to obtain the
55   * current time rather than a static method. This can simplify testing.
56   * <p>
57   * Best practice for applications is to pass a {@code Clock} into any method
58   * that requires the current instant. A dependency injection framework is one
59   * way to achieve this:
60   * <pre>
61   *  public class MyBean {
62   *    private Clock clock;  // dependency inject
63   *    ...
64   *    public void process(LocalDate eventDate) {
65   *      if (eventDate.isBefore(LocalDate.now(clock)) {
66   *        ...
67   *      }
68   *    }
69   *  }
70   * </pre>
71   * This approach allows an alternate clock, such as {@link #fixed(Instant, ZoneId) fixed}
72   * or {@link #offset(Clock, Duration) offset} to be used during testing.
73   * <p>
74   * The {@code system} factory methods provide clocks based on the best available
75   * system clock This may use {@link System#currentTimeMillis()}, or a higher
76   * resolution clock if one is available.
77   *
78   * <h3>Specification for implementors</h3>
79   * This abstract class must be implemented with care to ensure other operate correctly.
80   * All implementations that can be instantiated must be final, immutable and thread-safe.
81   * <p>
82   * The principal methods are defined to allow the throwing of an exception.
83   * In normal use, no exceptions will be thrown, however one possible implementation would be to
84   * obtain the time from a central time server across the network. Obviously, in this case the
85   * lookup could fail, and so the method is permitted to throw an exception.
86   * <p>
87   * The returned instants from {@code Clock} work on a time-scale that ignores leap seconds.
88   * If the implementation wraps a source that provides leap second information, then a mechanism
89   * should be used to "smooth" the leap second, such as UTC-SLS.
90   * <p>
91   * Implementations should implement {@code Serializable} wherever possible and must
92   * document whether or not they do support serialization.
93   */

208     /**
209      * Obtains a clock that returns instants from the specified clock truncated
210      * to the nearest occurrence of the specified duration.
211      * <p>
212      * This clock will only tick as per the specified duration. Thus, if the duration
213      * is half a second, the clock will return instants truncated to the half second.
214      * <p>
215      * The tick duration must be positive. If it has a part smaller than a whole
216      * millisecond, then the whole duration must divide into one second without
217      * leaving a remainder. All normal tick durations will match these criteria,
218      * including any multiple of hours, minutes, seconds and milliseconds, and
219      * sensible nanosecond durations, such as 20ns, 250,000ns and 500,000ns.
220      * <p>
221      * A duration of zero or one nanosecond would have no truncation effect.
222      * Passing one of these will return the underlying clock.
223      * <p>
224      * Implementations may use a caching strategy for performance reasons.
225      * As such, it is possible that the start of the requested duration observed
226      * via this clock will be later than that observed directly via the underlying clock.
227      * <p>
228      * The returned implementation is immutable, thread-safe and {@code Serializable}
229      * providing that the base clock is.
230      *
231      * @param baseClock  the base clock to base the ticking clock on, not null
232      * @param tickDuration  the duration of each visible tick, not negative, not null
233      * @return a clock that ticks in whole units of the duration, not null
234      * @throws IllegalArgumentException if the duration is negative, or has a
235      *  part smaller than a whole millisecond such that the whole duration is not
236      *  divisible into one second
237      * @throws ArithmeticException if the duration is too large to be represented as nanos
238      */
239     public static Clock tick(Clock baseClock, Duration tickDuration) {
240         Objects.requireNonNull(baseClock, "baseClock");
241         Objects.requireNonNull(tickDuration, "tickDuration");
242         if (tickDuration.isNegative()) {
243             throw new IllegalArgumentException("Tick duration must not be negative");
244         }
245         long tickNanos = tickDuration.toNanos();
246         if (tickNanos % 1000_000 == 0) {
247             // ok, no fraction of millisecond
248         } else if (1000_000_000 % tickNanos == 0) {
249             // ok, divides into one second without remainder
250         } else {
251             throw new IllegalArgumentException("Invalid tick duration");
252         }
253         if (tickNanos <= 1) {
254             return baseClock;
255         }
256         return new TickClock(baseClock, tickNanos);
257     }

284      * <p>
285      * This clock wraps another clock, returning instants that are later by the

291      * Passing zero will return the underlying clock.
292      * <p>

295      *

298      * @return a clock based on the base clock with the duration added, not null

302         Objects.requireNonNull(offsetDuration, "offsetDuration");
303         if (offsetDuration.equals(Duration.ZERO)) {
304             return baseClock;
305         }

545     /**
546      * Implementation of a clock that adds an offset to an underlying clock.
547      */
548     static final class TickClock extends Clock implements Serializable {
549         private static final long serialVersionUID = 6504659149906368850L;
550         private final Clock baseClock;
551         private final long tickNanos;
552 
553         TickClock(Clock baseClock, long tickNanos) {
554             this.baseClock = baseClock;
555             this.tickNanos = tickNanos;
556         }
557         @Override
558         public ZoneId getZone() {
559             return baseClock.getZone();
560         }
561         @Override
562         public Clock withZone(ZoneId zone) {
563             if (zone.equals(baseClock.getZone())) {  // intentional NPE
564                 return this;
565             }
566             return new TickClock(baseClock.withZone(zone), tickNanos);
567         }
568         @Override
569         public long millis() {
570             long millis = baseClock.millis();
571             return millis - Jdk8Methods.floorMod(millis, tickNanos / 1000_000L);
572         }
573         @Override
574         public Instant instant() {
575             if ((tickNanos % 1000_000) == 0) {
576                 long millis = baseClock.millis();
577                 return Instant.ofEpochMilli(millis - Jdk8Methods.floorMod(millis, tickNanos / 1000_000L));
578             }
579             Instant instant = baseClock.instant();
580             long nanos = instant.getNano();
581             long adjust = Jdk8Methods.floorMod(nanos, tickNanos);
582             return instant.minusNanos(adjust);
583         }
584         @Override
585         public boolean equals(Object obj) {
586             if (obj instanceof TickClock) {
587                 TickClock other = (TickClock) obj;
588                 return baseClock.equals(other.baseClock) && tickNanos == other.tickNanos;
589             }
590             return false;
591         }
592         @Override
593         public int hashCode() {
594             return baseClock.hashCode() ^ ((int) (tickNanos ^ (tickNanos >>> 32)));
595         }
596         @Override
597         public String toString() {
598             return "TickClock[" + baseClock + "," + Duration.ofNanos(tickNanos) + "]";
599         }
600     }
